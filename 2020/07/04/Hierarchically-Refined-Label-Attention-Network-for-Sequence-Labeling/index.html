Hierarchically-Refined Label Attention Network for Sequence Labeling

针对传统序列标注问题的推断层中使用 CRF 导致计算效率较低，使用 Softmax 导致忽略了标签之间的关系的情况，文中设计了 Label Attention Network (LAN)。通过将 Bi-LSTM 与 LAN 相融合，进行序列标签推断。

2. 研究领域与目标

2.1 研究领域

  • 序列标注

2.2 研究目标

  • 针对传统序列标注问题的推断层中使用 CRF 导致计算效率较低，使用 Softmax 导致忽略了标签之间的关系的情况，设计一种网络以兼顾效率与准确率

3. 个人总结

3.1 优点

  • 依靠多层叠加编码标签与标签之间的依赖

4. 问题定义

  • 输入为自然语言序列 \(x = \{ x_1, \dots , x_n \}\)
  • 输出为标签序列 \(y = \{ y_1, \dots , y_n \}\)，其中 \(y_i \in \{ l_1, \dots, l_{|L|} \}\)

5. 方法

5.1 Baseline

5.1.1 词表示层（word representation layer）

  • 将字符嵌入与词嵌入 concatenate 起来
    • 其中针对字符嵌入，使用 Bi-LSTM

5.1.2 序列表示层（sequence representation layer）

  • Bi-LSTM，获得 \(\{ h_1, \dots , h_n \}\)

5.1.3 推断层（inference layer）

  • CRF，其中 \(\mathcal{Y_x}\) 代表在序列 \(x\) 的情况下所有可能的标签序列 \[P(y|x) = \frac{\prod_{t=1}^n \exp(W_{y_{t-1},y_t}^T h_t + b_{y_{t-1},y_{t}})}{ \sum_{y' \in \mathcal{Y_x}} \prod_{t=1}^n \exp(W_{y'_{t-1},y'_t}^T h_t + b_{y'_{t-1},y'_{t}}) }\]

  • Softmax，与 CRF 相比，有两点不同：
    • 模型参数与当前时刻的标签无关，且与上一时刻的标签也无关
    • Softmax 在当前时刻的标签上归一化，而 CRF 在所有的标签搜索空间进行归一化 \[ y_i = \text{softmax}(W^T h_i+b) \]

5.2 标签注意力网络（label attention network, LAN）

  • 针对序列表示层和推断层进行修改和融合，去除推断层，将推断融合到序列表示中
  • 整体结构为：
    • 词表示层
    • 多层 Bi-LSTM-LAN

Framework

  • 每个标签使用标签嵌入，整合使用 \(x^l \in \mathbb{R}^{|L| \times d_h}\) 进行表示
  • Bi-LSTM-LAN
    • Bi-LSTM Encoding Sublayer，获得 \(H^w\)
    • Label-Attention Inference Sublayer
      • 使用多头注意力机制（相比于 Transformer 中的多头注意力机制，没有使用 \(W^O\)，且增加了残差连接），其中 \(Q = H^w, K,V=x^l\) \[ H^l = \text{concat}(head_1, \dots , head_k) + H^w \\ head_i = \text{attention}(QW_i^Q, KW_i^K, VW_i^V) = \alpha VW_i^V \]
      • 最终使用 \(H^w\) 与 \(H^l\) 的 concatenation 作为 Bi-LSTM-LAN 的输出表示 \(H=[H^w;H^l]\)
  • 输出
    • 设置最后一层的 Bi-LSTM-LAN 头数为 1，且使用 Label-Attention Inference Sublayer 中多头注意力机制里句子和标签对应的权重 \(\alpha \in \mathbb{R}^{n \times |L|}\)，作为模型预测的语句每个位置的标签的概率
    • 使用句子中每个位置概率最高的标签作为模型在该位置的预测标签
  • 一层的 Bi-LSTM-LAN 与单层的 Bi-LSTM-softmax 相同

5.3 实验设置

  • 100 维的 GloVe 作为词嵌入
